## 📌 Project Overview

Omega Tech Innovations is a modern, responsive website developed to showcase technology services, innovations, and digital solutions.  
This project was designed and built as a real-world web development project focusing on clean UI, performance, and scalable component architecture.

---

## 🚀 Features

- Responsive modern user interface
- Component-based architecture
- Optimized performance
- Clean navigation system
- Professional branding layout
- Mobile-friendly design

---

## 🛠️ Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- ShadCN UI
- Git & GitHub
- Vercel (Deployment)

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone repository
git clone https://github.com/Nitheesh-raj/omega-tech-innovations.git

# Navigate into project folder
cd omega-tech-innovations

# Install dependencies
npm install

# Run development server
npm run dev
