

# Omega Tech — Informational Website

A modern, techy, dark-themed website showcasing Omega Tech's courses, internships, and services.

## Design Direction
- **Dark theme** with neon accent colors (electric blue/cyan gradients), subtle glowing effects, and a futuristic tech aesthetic
- Bold typography, smooth scroll animations, and tech-inspired visual elements
- Fully responsive for mobile, tablet, and desktop

## Pages & Sections

### 1. Home / Landing Page
- **Hero Section** — Bold headline "Innovate. Learn. Lead." with animated background (gradient mesh or particles effect), CTA buttons for "Explore Courses" and "Apply for Internship"
- **About Section** — Brief intro to Omega Tech with vision & mission highlights
- **Why Choose Us** — Icon cards showcasing key benefits (Industry Curriculum, Live Projects, Certification, Placement Assistance, Expert Mentors, Affordable Fees)
- **Featured Courses Preview** — Highlight top course categories with links to the full courses page
- **Who Can Join** — Visual cards for different student types (School, Diploma, Engineering, Graduates, Professionals, Entrepreneurs)
- **Training Modes** — Display the 5 training modes (Offline, Online, Hybrid, Corporate, College Tie-ups)
- **Contact Section** — Address, email, phone number, and a mailto-based inquiry form

### 2. Courses Page
- Organized by category with tabs or filter navigation:
  - AI & Machine Learning (8 courses)
  - Data Science & Analytics (6 courses)
  - Robotics & Automation (6 courses)
  - Software Development (6 courses)
  - Cybersecurity (5 courses)
  - Cloud & DevOps (5 courses)
  - Emerging Technologies (5 courses)
- Each course shown as a card with name and category icon

### 3. Internship Program Page
- Overview of the internship program and its benefits
- Internship domains listed with icons
- Duration options (1, 2, 3, 6 months) displayed as comparison cards
- What's included checklist (project-based training, mentoring, certificates, etc.)
- Benefits section with highlights like live projects, GitHub portfolio, placement support

### 4. Placement & Career Support Page
- Services: Resume Building, Mock Interviews, LinkedIn Optimization, Career Counseling, Job Referrals, Startup Guidance
- Displayed as feature cards with icons

### Navigation & Footer
- **Sticky header** with Omega Tech logo, navigation links, and a "Contact Us" CTA button
- **Footer** with quick links, contact info, and social media placeholders
